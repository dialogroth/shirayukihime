## 1. ゲーム概要

| 項目 | 内容 |
|------|------|
| ゲーム名 | 白雪姫のアップルルーレット |
| 参加人数 | 4〜9名 |
| テーマ | 白雪姫をモチーフにした正体隠匿・リンゴ交換ゲーム |
| 目的 | 各陣営の勝利条件を満たすこと |

各プレイヤーは**役職カード**と**リンゴ**を1つ持ってゲームを開始します。リンゴの中には複数の**毒リンゴ**が含まれており、特定の役職を除いて外見からは判別できません。プレイヤーはカードの効果や役職能力を駆使しながら、エンディングフェイズに毒リンゴを持たない状態を目指します。

## 2. コンポーネント

### 2-1. リンゴ

| 種類 | 説明 |

## 3. 役職一覧

| 役職名 | 陣営 | 特殊情報 | 手番能力 | 固有能力 |
|-------|------|---------|---------|---------|
| 白雪姫 | 白雪姫陣営 | なし | なし | なし |
| 女王 | 女王陣営 | なし | エンディング特権あり（後述） | なし |

| グレイ | 白雪姫陣営 | なし | **役職能力を発動すると、次の自分の手番まで交換対象にならない（1ゲームに1回）** | なし |
| ネイビー | 女王陣営 | なし | なし | **リンゴは好き？・きのこは好き？への回答でYes/Noを自由に選べる（嘘がつける）** |

- **原則として自分以外の役職は見ることができない**
- グリーンには白雪姫の役職情報がゲーム開始時から開示される
- ブラックには全リンゴの毒/通常の配置情報がゲーム開始時から開示される
- 他プレイヤーへの役職情報は、カード効果・役職能力によってのみ開示される場合がある

## 4. 陣営と勝利条件

### 4-1. 陣営構成

| 陣営名 | 所属役職 |
|-------|---------|
| 白雪姫陣営 | 白雪姫、グリーン、ブラウン、グレイ、ライト |
| 女王陣営 | 女王、ブラック、ネイビー |
| 第三陣営 | ロゼ |

### 4-2. 勝利条件

| 陣営 | 勝利条件 |
|-----|---------|
| 白雪姫陣営 | **白雪姫が生きている** かつ **ロゼが死んでいる** |
| 女王陣営 | **白雪姫が死んでいる** |
| 第三陣営（ロゼ） | **白雪姫が生きている** かつ **ロゼが生きている** |

> **注意：** 各ゲームで必ず1つの陣営のみが勝利する。複数陣営が同時に勝利することはない。

> **ロゼが存在しない場合：** ロゼが選択されていないゲームでは、第三陣営は参加していない。この場合「ロゼが死んでいる」条件を自動的に満たしたものとみなす。したがって白雪姫が生存すれば白雪姫陣営の勝利となり、白雪姫が死亡すれば女王陣営の勝利となる。

### 5-2. UI制約

- 回答したプレイヤーの横に、回答結果（リンゴ好き / リンゴ嫌い / きのこ好き / きのこ嫌い）をアイコンまたはラベルで表示する
- 一度記録された結果はゲーム終了まで画面上に残す

## 6. スタート画面・ルーム仕様

### 6-1. スタート画面

ゲーム起動時に最初に表示される画面。**まずユーザー名を入力し**、その後に以下の2つのボタンを表示する。

| 項目 | 内容 |
|-----|------|
| ユーザー名入力フィールド | プレイヤー名を入力する。ゲーム中・ルーム一覧での表示名として使用される |
| 部屋を作るボタン | ルーム作成画面へ遷移。ユーザー名未入力の場合は非活性 |
| 部屋に入るボタン | ルーム参加画面へ遷移。ユーザー名未入力の場合は非活性 |

- ユーザー名は1文字以上・適切な上限文字数（例：12文字）を設ける
- 同一ルーム内で重複するユーザー名は使用不可とし、エラーメッセージを表示する

---

### 6-2. ルーム作成画面

ホストプレイヤーが以下の設定を行い、ルームを作成する。

#### 設定項目

**① 毒リンゴの枚数**

- 参加人数に応じた枚数を数値で設定する（スライダーまたはセレクトUI）
- 最小1枚〜最大（参加人数 − 1）枚の範囲で選択可能
- 参考初期値（ゲームバランス推奨値）：

| 参加人数 | 推奨毒リンゴ枚数 |
|---------|--------------|
| 4名 | 1〜2枚 |
| 5名 | 2枚 |
| 6名 | 2枚 |
| 7名 | 2〜3枚 |
| 8名 | 3枚 |
| 9名 | 3枚 |

**② 役職の組み合わせ**

- 使用する役職セットをチェックボックスで選択する
- 役職ごとに「使用する / しない」を切り替え可能
- 選択された役職の数 ＝ 参加人数となるように制約を設ける（過不足がある場合はエラー表示）
- 参加人数ごとの推奨役職セットをプリセットとして用意する（任意）

| 参加人数 | 推奨役職セット例 |
|---------|---------------|
| 4名 | 白雪姫・女王・グリーン・ブラック |
| 5名 | 白雪姫・女王・グリーン・ブラック・ライト |
| 6名 | 白雪姫・女王・グリーン・ブラック・ブラウン・ライト |
| 7名 | 白雪姫・女王・グリーン・ブラック・ブラウン・ライト・ロゼ |
| 8名 | 白雪姫・女王・グリーン・ブラック・ブラウン・ネイビー・ライト・ロゼ（グレイは任意） |
| 9名 | 白雪姫・女王・グリーン・ブラック・ブラウン・グレイ・ネイビー・ライト・ロゼ（全役職） |

> **補足：** 役職は全9種（白雪姫・女王・グリーン・ブラック・ブラウン・グレイ・ネイビー・ロゼ・ライト）あり、参加人数に合わせて使用する役職を選択する。9名の場合はすべての役職を使用する。8名以下の場合はいずれかの役職を除外する。

**③ カードの枚数**

- 各カード種別の枚数を個別に設定できる
- 初期値は以下のデフォルト設定を使用する：

| カード名 | デフォルト枚数 | 設定可能範囲 |
|---------|-------------|------------|
| リンゴは好き？ | 5枚 | 0〜10枚 |
| きのこは好き？ | 5枚 | 0〜10枚 |
| いただきます。 | 3枚 | 0〜5枚 |
| アップルルーレット！ | 1枚 | 0〜3枚 |
| アップルルーレット！！ | 1枚 | 0〜3枚 |
| アップルルーレット！！！ | 1枚 | 0〜3枚 |
| 包丁 | 1枚 | 0〜3枚 |
| 呪いの指輪 | 1枚 | 0〜3枚 |
| 毒の櫛 | 1枚 | 0〜3枚 |
| 騎士 | 1枚 | 0〜3枚 |
| ガード | 1枚 | 0〜3枚 |
| ロープ | 1枚 | 0〜3枚 |
| プレゼント交換 | 1枚 | 0〜3枚 |

- デフォルト設定に戻すボタンを設置する

#### ルーム作成後の動作

- 設定完了後「ルームを作成する」ボタンを押すと、**ルームコード**（数字6桁）が発行される
- ルームコードを他プレイヤーに共有することで参加できる
- ルームコードは**ゲーム中も常に画面上部に表示する**（切断復帰時にコードを確認できるようにするため）
- ホストはルーム待機画面で参加者の入室状況を確認できる
- **現在の参加人数が選択した役職数と一致する**と「ゲーム開始」ボタンが活性化される
- ホストのみがゲームを開始できる

---

### 6-3. ルーム参加画面

- ルームコード入力フィールドを表示する
- 「参加する」ボタン押下でルーム待機画面へ遷移する
- 存在しないコードや満員のルームにはエラーメッセージを表示する

---

### 6-3-2. ルーム復帰画面

- スタート画面に「部屋に復帰する」ボタンを配置する
- ルームコード入力フィールドを表示する（ユーザー名はスタート画面で入力済みのものを使用）
- 「復帰する」ボタン押下で、入力されたルームコードとユーザー名に一致する既存プレイヤーを検索する
- 一致するプレイヤーが見つかった場合：
  - ルームがゲーム中（IN_GAME）であれば、WebSocket再接続してゲーム画面に直接遷移する
  - ルームが待機中であれば、ルーム待機画面に遷移する
- 一致するプレイヤーが見つからない場合はエラーメッセージを表示する

---

### 6-4. ルーム待機画面（共通）

- 現在の参加者一覧（プレイヤー名）を表示する
- ルームコードを画面上に常時表示する（共有用）
- ホストには「ゲーム開始」ボタンを表示し、**現在の参加人数が選択した役職数と一致する**場合に活性化する
- ゲスト（非ホスト）には「ゲーム開始待ち」の表示をする
- **「ルームから抜ける」ボタン**を配置する：
  - **ホストが押した場合：** 確認ダイアログ表示後、ルームを解散する。他の全参加者に「ルームが解散されました」と通知オーバーレイを表示し、「ホームへ戻る」ボタンでスタート画面に遷移させる
  - **ゲスト（非ホスト）が押した場合：** 自分だけがルームから退出し、スタート画面に戻る

#### 6-4-1. 座席配置（円形レイアウト）

- ホストが設定した**役職数（＝部屋の定員）分の席**を、円形に均等に配置する
- 席には**プレイ順番（1, 2, 3, ...）**を表示する
- 12時の位置が**1番席（スタートプレイヤーの席）**とし、時計回りに2番、3番...と配置する
- プレイヤーは入室順に12時の位置から時計回りに配置される（ホストが最初に12時の位置に入る）
- 空席は「空席」と表示する

#### 6-4-2. スタートプレイヤーとプレイ順

- **スタートプレイヤー：** 12時の位置（1番席）に座っているプレイヤーで固定
- **プレイ順番：** 時計回り（1番席 → 2番席 → 3番席 → ... → 最後の席 → 1番席に戻る）

#### 6-4-3. 座席の並び替え（ホスト専用）

- **ホストのみ**が全プレイヤーの座席位置を**ドラッグアンドドロップ**で入れ替えることができる
- ドラッグアンドドロップ操作中の様子（どのプレイヤーをドラッグしているか、どの席の上にいるか）は、**リアルタイムで全参加者に配信される**
- ドロップ確定時に座席の入れ替えが確定し、全プレイヤーの画面に反映される
- これにより、ホストはゲーム開始前にプレイ順を自由に調整できる
- **「ランダム配置」ボタン：** ホストのみ使用可能。押下すると全参加者の座席をランダムにシャッフルし、結果を全員に配信する

#### 6-4-4. 満席時の参加拒否

- ルームの定員（＝役職数）に達した場合、それ以降にルームコードで参加しようとしたプレイヤーには**「満席です」**とエラーメッセージを表示し、参加を拒否する
- 満席状態でも「部屋に復帰する」による既存プレイヤーの再接続は可能

---

### 6-5. 再ゲーム仕様

ゲーム終了（勝敗判定完了）後、同じメンバーでもう一度プレイできる仕組みを設ける。

- 勝敗画面に**「もう一度遊ぶ」**ボタンを表示する
- ホストが「もう一度遊ぶ」を押すと、切断していないプレイヤーはルームIDを再入力せずそのままルーム待機画面に戻る
- ゲーム設定（毒リンゴ枚数・役職・カード枚数）は前ゲームの設定をそのまま引き継ぐ。設定を変更したい場合はルームから抜けて新しい部屋を作り直す
- 「ルームから抜ける」ボタンも配置し、スタート画面に戻れるようにする（ホストの場合はルーム解散）

#### 再ゲーム時の切断プレイヤーの扱い

- 前ゲーム中に切断により死亡扱いになったプレイヤーは、**ルーム待機画面に切断状態（空席）で表示される**
- 1分以内に再接続した場合はそのまま同じ席でゲームに参加できる
- 再接続しないままホストがゲームを開始した場合、そのプレイヤーは参加できずルームから除外される
- 除外により参加人数が役職数と一致しなくなった場合はゲーム開始不可。ルームを解散して新しい部屋を作り直す必要がある

## 7. ゲームの準備

1. **役職カードの配布：** 参加人数に応じた役職カードをシャッフルし、各プレイヤーに1枚ずつ裏向きで配る（自分の役職のみ閲覧可能）
2. **リンゴの配布：** 通常リンゴと毒リンゴを混ぜてランダムに各プレイヤーへ1個ずつ配る。配布後、**自分のリンゴが毒か通常かを本人のみに通知する**
3. **特殊情報の開示：**
   - グリーン役のプレイヤーに「白雪姫が誰か」をシステムが通知する
   - ブラック役のプレイヤーに「毒リンゴの位置（誰が持っているか）」をシステムが通知する
4. **手札の配布：** 山札をシャッフルし、各プレイヤーに1枚ずつ配る
5. **最初の手番プレイヤーの決定：** ランダムに1人を選び、その人から時計回りに手番を行う

## 8. ゲームの流れ

### 8-1. フェイズ構成

```
[準備フェイズ]
      ↓
[ストーリーフェイズ]（山札がなくなるまで繰り返し）
      ↓
[最後の手番フェイズ]（山札が残り1枚になった次のプレイヤーから一周）
      ↓
[エンディングフェイズ]
      ↓
[勝敗判定]
```

---

### 8-2. ストーリーフェイズ

手番プレイヤーは以下の**5つの行動のうち1つ**を選択して実行します。

#### ① 山札を引く

- 山札から1枚引き、手札が一時的に2枚になる
- **山札の最後の1枚は引くことができない**（残り1枚になった時点で最後の手番フェイズに移行し、山札を引く行動は選択不可となる）
- **引いた後は必ずどちらかのカードを使用または捨て山に廃棄し、手番終了時の手札を1枚に戻す**
- **カードを使用する場合：** カードのタイトルを全員に読み上げ、効果を発動後、捨て山（公開）に置く
- **カードを廃棄する場合：** 効果を使わずそのまま捨て山に置く
- **例外：** 「呪いの指輪」は廃棄・使用ができないため、呪いの指輪を手札に持ちながら山札を引いた場合は、新たに引いたカードを使用するか廃棄するかを選択する（呪いの指輪は手元に残り続ける）
- カードの効果は「9. カード一覧・効果」を参照

#### ② 手札の交換

- 手番プレイヤーが任意のプレイヤー1名を指定し、そのプレイヤーと手札を交換する
- 交換は非公開（中身を見せずに）で行う
- **呪いの指輪の受け渡し：** この交換によって相手に呪いの指輪を渡した場合、渡された相手は次の自分の手番から呪いの指輪の効果対象になる

#### ③ リンゴの交換

- 自分のリンゴと任意のプレイヤー1人のリンゴを交換する
- 交換は非公開で行う（ただし「一度判明したリンゴ」は判別可能なまま）

#### ④ 自分のリンゴの確認

- 自分のリンゴが毒か通常かを**自分だけ**確認できる
- 他プレイヤーには見せない（システムが該当プレイヤーのみに情報を表示）

#### ⑤ キャラクターの能力開示

- 手番で使用できる特殊能力を持つ役職のプレイヤーは、その能力を宣言・発動できる

**現在定義されている手番能力：**

| 役職 | 能力内容 |
|-----|---------|
| グレイ | 役職能力を宣言して発動する。発動から次の自分の手番が来るまでの間、他プレイヤーのリンゴ交換・手札交換の対象に指定されない。ストーリーフェイズ・最後の手番フェイズのどちらでも発動可能。発動した保護効果はフェイズをまたいで持続する。**保護中はライトの能力でリンゴ交換の対象に指定された場合も無効となる（グレイの保護が優先）。ただし包丁によるリンゴの公開（交換ではない）は保護対象外。1ゲームに1回のみ使用可能。保護発動中にロープでスキップされた場合、スキップされた手番は「次の手番」とみなさないため保護は解除されず、次の（スキップされない）手番まで継続する** |
| ライト | 自分の役職カードを全員に公開する（役職が恒久的に開示される）。その代わりに、任意の2名を指定してその2名のリンゴを交換させる。ストーリーフェイズ・最後の手番フェイズのどちらでも使用可能。**1ゲームに1回のみ使用可能** |

### 8-3. 最後の手番フェイズ

- **移行条件：** 山札が残り1枚になった時点でこのフェイズに移行する
- **最後の1枚：** 山札の最後の1枚は引くことができず、いただきますで捨てることもできない（残り1枚になった時点でフェイズが移行し、山札は触れない状態となる）
- **開始プレイヤー：** 山札を残り1枚にしたプレイヤーの次のプレイヤー（時計回り）
- **範囲：** 開始プレイヤーから時計回りに全員が1回ずつ手番を行う（合計1周）
- **制限：** 「最後の手番フェイズ専用カード」はこのフェイズでのみ使用可能

#### 呪いの指輪の判定

- **判定タイミング：** 最後の手番フェイズ中に、呪いの指輪を手札に持つプレイヤーに手番が回ってきた瞬間に、そのプレイヤーは**即座に死亡**する
- 手番が回ってきた時点で判定されるため、該当プレイヤーはその手番で一切の行動を行えない
- 最後の手番フェイズに移行しただけでは死亡しない。手番が回ってくるまでは生存している
- 呪いの指輪は手札から捨てることができないため、ストーリーフェイズ中に処分することはできない（②手札の交換・プレゼント交換で他プレイヤーに渡すことのみ可能）

#### 最後の手番フェイズにおける死亡後の共通挙動

毒の櫛・呪いの指輪によって最後の手番フェイズ中に死亡したプレイヤーには、以降のフェイズで以下の制約が共通して適用される：

- **手番スキップ：** 死亡したプレイヤーの残り手番（最後の手番フェイズ内）は自動的にスキップされる
- **交換行動の無効：** 他プレイヤーが死亡プレイヤーをリンゴ交換・手札交換の対象に指定できない。また死亡プレイヤー自身も交換行動を行えない
- **アップルルーレットの対象外：** アップルルーレット系カード使用時、死亡プレイヤーのリンゴはローテーションから除外される（死亡プレイヤーの座席を飛ばして隣のプレイヤーへリンゴが移動する）

---

### 8-4. エンディングフェイズ

処理は以下の順序で行う：

1. **女王の特権（リンゴ公開と条件付き交換）**
   - 女王は自分のリンゴをまず全員に公開する
   - 自分のリンゴが**毒リンゴの場合：** 生存している任意のプレイヤー1人を指定し、そのプレイヤーとリンゴを交換する
   - 自分のリンゴが**通常リンゴの場合：** 交換は行えない（強制的にスキップ）
   - **女王が既に死亡している場合（毒の櫛・呪いの指輪による）：** 女王の特権は発動しない（スキップ）

2. **全員のリンゴ公開**
   - 残りの生存プレイヤー全員のリンゴを表にする
   - 毒リンゴを持つプレイヤーは死亡する

3. **勝敗判定**
   - 「4-2. 勝利条件」に基づいて各陣営の勝敗を確認する

- **使用フェイズ：** ストーリーフェイズ、最後の手番フェイズ
- **効果：** 任意のプレイヤー1名を指定し、そのプレイヤーのリンゴ（毒か通常か）を**全員に公開**する
- **記録：** 公開されたリンゴ情報を当該プレイヤーの横に永続表示する（移動後も追従）

#### ロープ（1枚）

- **使用フェイズ：** ストーリーフェイズ、最後の手番フェイズ
- **効果：** 任意のプレイヤー1名を指定し、そのプレイヤーの**次の手番を強制的にスキップ**させる
- **補足：** スキップされた手番では行動を一切行えない。スキップ状態は次の手番1回のみに適用される
- **フェイズ持続：** スキップが未消化のままストーリーフェイズが終了し最後の手番フェイズへ移行した場合、スキップは持続する。最後の手番フェイズでの当該プレイヤーの手番が消化された時点で解除される
- **グレイの保護との関係：** グレイが保護を発動中にロープでスキップされた場合、スキップされた手番はグレイの行動機会とみなさないため保護は解除されない。次の（スキップされない）手番が来た時点で保護が解除される

#### プレゼント交換（1枚）

- **使用フェイズ：** ストーリーフェイズ、最後の手番フェイズ
- **効果：** 任意の2名のプレイヤーを指定し、その2名の**手札**を交換させる
- **呪いの指輪の受け渡し：** この交換によって呪いの指輪が別のプレイヤーへ渡った場合、渡された相手は次の自分の手番から呪いの指輪の効果対象になる
- **グレイの保護との関係：** グレイが保護を発動中の場合、プレゼント交換の対象に指定されても手札の交換は行われない（グレイの保護は手札交換にも適用される）。グレイ以外のもう1名との交換も不成立となる

### 9-3. 受動・特定フェイズ専用カード（手元に持つことで機能するもの）

#### 呪いの指輪（1枚）

- **効果の発動フェイズ：** 最後の手番フェイズ中、手番が回ってきた時。ストーリーフェイズでは効果は発動しないが、引いた場合は強制的に手札に入り捨てることができない
- **効果：** 最後の手番フェイズ中に、このカードを手札に持つプレイヤーに手番が回ってきた瞬間に、そのプレイヤーは**即座に死亡**する
- **タイミング：** 手番が回ってきた瞬間に判定。該当プレイヤーはその手番で一切の行動を行えない。最後の手番フェイズに移行しただけでは死亡しない
- **捨て不可：** フェイズを問わず手札から捨てることができない。また呪いの指輪そのものを使用することもできない
- **ストーリーフェイズ中の挙動：** 呪いの指輪を持っていても①山札を引く行動は可能。ただし呪いの指輪は廃棄できないため、新たに引いたカードを必ず使用または廃棄しなければならない（引いたカードを処理することで手札を1枚に戻す）
- **受け渡し：** ②手札の交換・プレゼント交換で他プレイヤーへ渡すことができる。ストーリーフェイズ中に渡すことで死亡を回避できる
- **死亡後の挙動：** 「8-3. 最後の手番フェイズにおける死亡後の共通挙動」を参照

#### 騎士（1枚）

- **使用フェイズ：** 手番での使用はない。最後の手番フェイズ中に受動的に機能する
- **効果：** **白雪姫が手札に持っている場合のみ**有効。毒の櫛の対象に指定されても死亡を免れる
- **消費：** 発動後もカードは手札に残る（消費しない）
- **補足：** 白雪姫以外のプレイヤーが持っていても効果はなく、毒の櫛で通常通り死亡する

#### ガード（1枚）

- **使用フェイズ：** エンディングフェイズ専用（女王の特権処理タイミングでのみ発動）
- **効果：** 女王が毒リンゴを持ちリンゴ交換対象として自分を指定してきた場合に、**自動的に**リンゴ交換を拒否する。プレイヤーの操作は不要
- **消費：** 発動後もカードは手札に残る（消費しない）
- **再指定：** ガードで拒否された場合、女王は別のプレイヤーへの再指定は**できない**（交換は不成立となる）

## 11. その他仕様・制約

### 11-1. 情報の非公開原則

| 情報 | 公開範囲 |
|------|---------|
| 役職 | 本人のみ（特殊情報は該当役職のみ） |
| 手札 | 本人のみ |
| 自分のリンゴの毒/通常 | **ゲーム開始時から本人に開示**（交換・移動後は再確認が必要） |
| 他プレイヤーのリンゴの毒/通常 | 原則非公開（包丁・エンディングフェイズを除く） |
| 捨て山 | 全員 |

### 11-2. 特殊情報の開示タイミング

- **全プレイヤーへの自分リンゴ開示：** ゲーム開始直後に各プレイヤーへ自分のリンゴが毒か通常かを通知する。ただしリンゴの交換・アップルルーレット後は情報が古くなるため、④自分のリンゴの確認アクションで再確認する必要がある
- グリーンへの白雪姫情報：**ゲーム開始直後**（準備フェイズ完了時）に自動通知
- ブラックへの毒リンゴ情報：**ゲーム開始直後**（準備フェイズ完了時）に自動通知
- アップルルーレット使用後も毒リンゴの位置はブラックに継続して開示されること

### 11-3. 山札の管理

- 山札が残り1枚になった瞬間に最後の手番フェイズへ移行する
- 移行のトリガーは「山札を引く」行動または「いただきます。」カード使用により山札が残り1枚になること
- いずれの場合も、山札を残り1枚にしたプレイヤーの次のプレイヤーから時計回りに最後の手番フェイズが始まる
- 山札の最後の1枚は引くことも、いただきますで捨てることもできない
- 最後の手番フェイズ中は山札に触れることができない

### 11-4. 手札の管理

- 手番以外では手札は常に**1枚**
- 自分の手番で①山札を引いた場合のみ一時的に2枚になるが、**必ずどちらかを使用または廃棄して手番終了時には1枚に戻す**
- 唯一の例外は「呪いの指輪」で、廃棄できないため引いた場合は手元に残り続ける

### 11-5. エラーおよび制約チェック

- 役職カードは参加人数に合わせて使用する役職セットを決定する（詳細は別途役職組み合わせ表を定義）

### 11-6. 手番タイムアウト時のカード処理

手番の3分タイムアウトが発生した場合、使用中または選択中のカードは以下のルールで処理する。

| 状況 | 処理 |
|-----|------|
| カードを使用中（対象プレイヤー選択中など）にタイムアウト | 使用中のカードを**捨て山**に送る |
| カードを引いた後、使用 / 廃棄を選択中にタイムアウト | 引いたカードを**捨て山**に送る |
| 手札に**呪いの指輪**を持ちながら山札を引いた後にタイムアウト | 呪いの指輪は捨て山に送れないため、**新たに引いたカードを捨て山**に送る（呪いの指輪は手元に残る） |

### 11-6-2. 手番タイマーの表示と長考ボタン

#### タイマー表示

- 手番プレイヤーの手番開始時から**3分（180秒）のカウントダウンタイマー**を全プレイヤーに表示する
- タイマーは秒単位で更新し、残り時間を視覚的にわかりやすく表示する

#### 残り1分の警告

- 残り**60秒**になった時点で、手番プレイヤーに警告を表示する（タイマーの色を赤に変更、警告メッセージ表示など）
- 他プレイヤーにもタイマーが赤くなることで残り時間が少ないことがわかる

#### 長考ボタン

- 残り**15秒**になると、手番プレイヤーの画面に「長考」ボタンが出現する
- 「長考」ボタンを押すと、残り時間に**+2分（120秒）**が加算される
- **1ゲームにつき各プレイヤー1回のみ**使用可能
- 使用済みの場合はボタンが出現しない
- 長考ボタン使用時、全プレイヤーに「XXX が長考を使用しました」とログに表示する

### 11-7. プレイヤー切断時の仕様

#### ゲーム開始前（ルーム待機中）の切断

- 切断したプレイヤーの枠は**空席**に戻る
- 他のプレイヤーには切断が通知される（`PLAYER_DISCONNECTED`）
- 再接続した場合は同じ枠に戻る

#### ゲーム中の切断（白雪姫以外）

- 切断を検知してから**1分（60秒）以内に再接続**した場合はゲームを続行する
- **1分経過しても再接続がなかった場合、そのプレイヤーを死亡扱い**とし、以降の手番をスキップする
- 1分のカウント中に手番が回ってきた場合は、タイムアウト（1分）まで待ってからスキップする
- 死亡扱いになったプレイヤーのリンゴはエンディングフェイズで公開される

#### ゲーム中の切断（白雪姫）

- 白雪姫が切断した場合も**1分（60秒）以内に再接続**した場合はゲームを続行する
- **1分経過しても再接続がなかった場合、女王陣営の勝利としてゲームを終了する**
- 理由：白雪姫の存在がゲームの根幹に関わるため、復帰できない場合はゲーム進行を継続できない

## 12. 用語集

| 用語 | 定義 |
|-----|------|
| 山札 | ゲーム中に引くカードの束。中央に置かれる |
| 捨て山 | 使用済み・破棄されたカードの置き場。全員が閲覧可能 |
| 手札 | 各プレイヤーが手に持つカード。手番以外では常に1枚。自分の手番で山札を引いた際のみ一時的に2枚になる |
| リンゴ | 各プレイヤーが持つアイテム。通常と毒の2種 |
| 毒リンゴ | エンディングフェイズに持っていると死亡するリンゴ |
| 役職 | 各プレイヤーが持つキャラクター情報。陣営・能力・好みを規定する |
| 手番 | 各プレイヤーが行動を行う順番 |
| ストーリーフェイズ | 山札がなくなるまでの通常プレイフェイズ |
| 最後の手番フェイズ | 山札終了後の最終行動フェイズ |
| エンディングフェイズ | リンゴを公開し勝敗を決めるフェイズ |
| 時計回り | 座席を右方向に移動する順序 |
| 反時計回り | 座席を左方向に移動する順序 |
